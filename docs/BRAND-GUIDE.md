# Brand Guide

Concept: Scientific Humanist Lab.

AIMA should feel academic, rigorous, modern, thoughtful, and collaborative.

Use AIMA blue for primary actions, active states, and selective emphasis. Use deep blue for credibility accents. Use teal sparingly for scientific secondary cues.

Typography:

- Source Serif 4 for hero and editorial statements.
- Inter for navigation, headings, body, metadata, and controls.

Logo:

- Current files are placeholders because the allowed source logo was missing.
- Owner should provide `logo-full.svg`, `logo-wordmark.svg`, `logo-symbol.svg`, `favicon.svg`, and `default-og-image.png`.

Motion:

- Use restrained fade, hover, and reveal motion.
- Respect reduced motion.
- Avoid continuous or decorative animation.

Visuals:

- Prefer authentic research figures, diagrams, and community photography.
- Placeholders must be replaced before production launch.
